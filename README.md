# PDF Text Mover (PDF 标签文本区域迁移工具)

一个基于 Delphi 开发的 Win64 桌面应用程序，用于批量处理 PDF 标签文件，自动将右下角的单号文本区域迁移（移动）至顶部的空白框中。

## 功能特性

- **多模式文本区域定位**：
  - **智能识别模式 (推荐)**：免配置，程序会自动识别并移动右下角单号文本到顶部合适位置。
  - **百分比自适应模式**：以页面宽度和高度的百分比定义源区域与目标位置，完美适配不同尺寸的 PDF 页面。
  - **绝对坐标模式**：以点（pt / 磅）为单位指定精确的提取和目标坐标。
- **重建文本样式**：可自定义目标区域文本的字号。
- **擦除原始文本**：支持在迁移后将源区域的文字清除，保持页面整洁。
- **防文字重叠**：自适应避让重叠的背景或已有文本。
- **批量处理与拖拽支持**：支持选择多个 PDF 文件或直接拖拽文件/文件夹到主窗口进行批量处理。
- **核心组件加载校验**：启动时自动预加载并校验 PDF 渲染动态库，缺失时提供显眼的排查指引和下载链接。

## 环境要求

- **操作系统**：Windows 7 或更高版本 (64位)
- **核心依赖**：`pdfium.dll` (Win64 版本)

## 安装与运行

本程序为绿色版，无需安装。但在首次运行前，必须配置 PDFium 核心库：

1. **下载 DLL**：
   - 访问 [pdfium-binaries Releases](https://github.com/bblanchon/pdfium-binaries/releases)。
   - 下载适用于您的 Windows x64 版本的二进制包（例如 `pdfium-win-x64.tgz`）。
   - 解压并提取出 `pdfium.dll` 文件。
2. **放置 DLL**：
   - 将 `pdfium.dll` 复制并粘贴到 `PDFTextMover.exe` 程序的同级目录下。
3. **启动程序**：
   - 双击 `PDFTextMover.exe` 即可运行。若提示缺失核心库，请按照运行日志框内的提示信息下载并放置文件。

## 编译指南

如果您需要从源码重新编译此程序，请确保您的开发环境满足以下条件：

- 安装有 **Delphi XE3 或更高版本** (推荐 Delphi 11.x 社区版或专业版)。
- 打开 Delphi 命令行环境或在系统配置好 MSBuild。
- 直接双击运行项目根目录下的 `compile_mover.bat` 脚本，编译后的可执行文件将自动输出到 `bin/` 文件夹中。

## 目录结构说明

- `src/`：程序业务源码。
  - `forms/`：主窗体 UI 代码（`frmMoverMain.pas` 等）。
  - `services/`：PDF 处理核心服务类（`uPDFTextMover.pas`）。
- `Source/`：第三方 PDFium 绑定底层库（`PdfiumCore.pas`、`PdfiumLib.pas` 等）。
- `bin/`：编译输出的二进制可执行文件目录。
- `dcu/`：Delphi 编译产生的临时单元缓冲目录。
